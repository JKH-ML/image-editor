このツールは、ブラウザ上で即座に動作する包括的な画像編集スイートです。 **MediaPipe AI** と **HTML5 Canvas** を使用しており、すべての処理はデバイス上でローカルに行われます。写真はコンピュータの外に出ることはなく、完全なプライバシーが保証されます。

## ✨ 主な機能

- **🤖 AI 背景削除**: Google の MediaPipe Selfie Segmentation 技術を使用して背景を即座に削除します。プロ仕様のフェザリング処理により、エッジも非常に滑らかです。
- **🎨 フィルターと調整**: 明るさ、コントラスト、彩度、ぼかしをリアルタイムプレビューを見ながら微調整できます。
- **🎞️ アニメーション GIF メーカー**: 複数の画像から高品質な GIF を作成します。フレーム遅延やループ設定が可能です。
- **🧩 自由なコラージュ**: 複数の写真を自由に配置できます。**マウスホイールでの回転**、サイズ変更、レイヤーの重なり順変更に対応し、独自の構成を作成できます。
- **✂️ 高精度クロップ**: 自由な領域選択、または標準プリセット (1:1, 4:3, 16:9) を使用して切り抜けます。
- **🔗 画像結合**: 複数の画像を水平、垂直、またはグリッド形式で 1 つのファイルに結合します。
- **🔄 変形**: 90 度回転、および水平/垂直反転が可能です。
- **🌙 モダンな UI**: ダークモードを完全サポートした、クリーンな「Zinc」デザインシステムを採用。

## 🛠️ 技術スタック

- **AI エンジン**: MediaPipe Selfie Segmentation
- **コア技術**: HTML5 Canvas API, gifshot.js
- **スタイリング**: CSS3 (Custom Variables), Tailwind CSS
- **フレームワーク**: Vanilla JavaScript (ES6+)
- **デプロイ**: GitHub Pages (サーバーレス)

## 📖 使用方法

1.  **フィルター**: 画像をアップロードし、スライダーを動かして写真を補正します。
2.  **背景削除**: 「背景削除」タブで AI ボタンをクリックし、透明背景の PNG を作成します。
3.  **GIF メーカー**:
    - 「GIF メーカー」タブで複数の画像を選択します。
    - 矢印ボタンでフレームの順序を入れ替えます。
    - フレーム遅延 (ms) とループ回数を設定します。
    - 「GIF を作成する」をクリックして結果をダウンロードします。
4.  **コラージュ**:
    - 「コラージュ」タブで複数の写真を追加します。
    - ドラッグして移動し、**マウスホイールを回して回転**させます。
    - 各写真の右上の設定アイコンでサイズを調整します。
    - 画面キャプチャ (**Win + Shift + S**) を使用して保存してください。
5.  **結合/クロップ**: 専用のタブで目的に合わせた編集を行います。
