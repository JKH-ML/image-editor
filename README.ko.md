## ✨ 주요 기능

- **🤖 AI 배경 제거 (누끼 따기)**: Google의 MediaPipe 기술을 사용하여 인물을 정교하게 인식하고 배경을 즉시 제거합니다. 페더링 기술로 외곽선을 부드럽게 처리합니다.
- **🎨 필터 및 세밀 조정**: 밝기, 대비, 채도, 블러를 실시간 미리보기를 보며 미세하게 조정할 수 있습니다.
- **🎞️ 애니메이션 GIF 메이커**: 여러 이미지로 고품질 GIF를 제작하세요. 프레임 지연 및 반복 횟수 설정이 가능합니다.
- **🧩 자유형 콜라주**: 여러 사진을 자유롭게 배치하세요. **마우스 휠로 회전**, 크기 조절, 레이어 순서 변경 기능을 지원합니다.
- **✂️ 정밀 크롭**: 자유로운 영역 선택 및 주요 비율(1:1, 4:3, 16:9) 프리셋을 지원합니다.
- **🔗 이미지 병합**: 여러 이미지를 가로, 세로 또는 그리드 형태로 간편하게 합칠 수 있습니다.
- **🔄 이미지 변형**: 90도 회전 및 좌우/상하 반전 기능을 제공합니다.
- **🌙 모던 UI**: 다크 모드를 완벽 지원하는 깔끔한 Zinc 디자인 시스템을 적용했습니다.

## 🛠️ 기술 스택

- **AI 엔진**: MediaPipe Selfie Segmentation
- **핵심 기술**: HTML5 Canvas API, gifshot.js
- **스타일링**: CSS3 (Custom Variables), Tailwind CSS
- **프레임워크**: Vanilla JavaScript (ES6+)
- **배포**: GitHub Pages (Serverless)

## 📖 사용 방법

1.  **필터**: 이미지를 업로드하고 슬라이더를 움직여 사진을 보정합니다.
2.  **배경 제거**: '배경 제거' 탭에서 AI 버튼을 클릭해 투명 배경의 PNG를 만듭니다.
3.  **GIF 메이커**:
    - 'GIF 메이커' 탭에서 여러 이미지를 선택합니다.
    - 화살표 버튼으로 프레임 순서를 변경합니다.
    - 프레임 지연 시간(ms)과 반복 횟수를 설정합니다.
    - 'GIF 만들기'를 클릭하고 결과를 다운로드합니다.
4.  **콜라주**:
    - 사진을 여러 장 추가합니다.
    - 드래그로 이동하고, **마우스 휠을 돌려 회전**시킵니다.
    - 사진 우측 상단 설정 아이콘으로 크기를 조절합니다.
    - 화면 캡처(**Win + Shift + S**)를 이용해 결과물을 저장하세요.
5.  **병합/크롭**: 전용 탭에서 목적에 맞는 편집을 진행합니다.
