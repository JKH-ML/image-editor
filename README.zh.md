## ✨ 主要功能

- **🤖 AI 背景移除**：使用 Google 的 MediaPipe Selfie Segmentation 技术即时移除背景。采用专业的羽化处理，边缘质量极高。
- **🎨 滤镜与调整**：通过实时预览微调亮度、对比度、饱和度和模糊度。
- **🧩 自由拼贴**：自由排列多张照片。支持鼠标滚轮旋转、调整大小和图层排序，打造您独特的组合。
- **✂️ 精确裁剪**：支持自由裁切或使用标准预设（1:1, 4:3, 16:9）。
- **🔗 图像合并**：支持将多张图像水平、垂直或以网格形式合并为单个文件。
- **🔄 变换**：快速 90 度旋转以及水平/垂直翻转。
- **🌙 现代 UI**：简洁的 "Zinc" 设计系统，完美支持深色模式。

## 🛠️ 技术栈

- **AI 引擎**：MediaPipe Selfie Segmentation
- **核心**：HTML5 Canvas API
- **样式**：CSS3 (Custom Variables), Tailwind CSS
- **框架**：Vanilla JavaScript (ES6+)
- **部署**：GitHub Pages (无服务器)

## 📖 使用方法

1.  **滤镜**：上传图像并使用滑块增强您的照片。
2.  **背景移除**：前往“删除背景”选项卡，上传并点击 AI 按钮以获取透明 PNG。
3.  **拼贴**：
    - 在“拼贴”选项卡中添加多张照片。
    - 拖动以移动，**滚动鼠标滚轮以旋转**。
    - 使用每张照片上的设置图标调整大小。
    - 通过屏幕截屏 (**Win + Shift + S**) 进行保存。
4.  **合并/裁剪**：使用专用选项卡进行结构化编辑和组合。
